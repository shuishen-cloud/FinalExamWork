# 餐厅管理系统界面美化文档

## 1. 总体设计原则

- 现代简洁风格
- 统一的配色方案
- 响应式布局
- 用户友好的交互设计

## 2. 配色方案

### 主色调
- 主色: #2C3E50 (深蓝灰，用于标题栏、边框)
- 次色: #3498DB (亮蓝，用于按钮、链接)
- 辅助色: #E74C3C (红色，用于警告、删除)
- 强调色: #27AE60 (绿色，用于确认、成功)

### 背景色
- 主背景: #ECF0F1 (浅灰蓝)
- 面板背景: #FFFFFF (白色)
- 深色面板: #34495E (深蓝灰)

### 文字颜色
- 主文字: #2C3E50 (深蓝灰)
- 次要文字: #7F8C8D (中灰)
- 强调文字: #E74C3C (红色)

## 3. 字体规范

- 主字体: "Microsoft YaHei", "微软雅黑", sans-serif
- 标题字体大小: 16px (标题), 14px (主内容), 12px (辅助信息)
- 字重: Regular (正文), Bold (强调内容)

## 4. 组件规范

### 按钮
- 圆角: 6px
- 内边距: 8px 16px
- 主要按钮: 蓝色背景 (#3498DB)，白色字体
- 次要按钮: 灰色背景 (#95A5A6)，白色字体
- 危险按钮: 红色背景 (#E74C3C)，白色字体
- 成功按钮: 绿色背景 (#27AE60)，白色字体

### 面板
- 边框: 1px solid #BDC3C7
- 圆角: 8px
- 内边距: 15px
- 阴影: 0 2px 5px rgba(0,0,0,0.1)

### 表格
- 行高: 30px (比默认稍高)
- 行间距: 保持清晰
- 选中行: #D6EAF8 (浅蓝色背景)

### 输入框
- 边框: 1px solid #BDC3C7
- 圆角: 4px
- 内边距: 5px
- 聚焦边框: 2px solid #3498DB

## 5. 布局规范

- 主窗口: 1000x700px (保持现有尺寸)
- 标题栏: 30px 高
- 状态栏: 25px 高
- 内边距: 10px

## 6. 特殊效果

- 按钮悬停效果: 透明度调整 (0.8)
- 面板阴影效果: 统一使用 0 2px 5px rgba(0,0,0,0.1)
- 过渡动画: 简单的渐变效果 (如果可能)
