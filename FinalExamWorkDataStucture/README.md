# DataStructureFinalExamWork

## 《教学计划编制程序》设计说明


### 一、设计背景与目标
本程序针对大学专业教学计划编制的需求，解决“课程先修关系约束+学期学分上限”下的排课问题，支持两种排课策略（学分均匀/课程集中），最终输出结构化的教学计划文件。


### 二、核心设计思路
#### 1. 数据模型设计
- **课程结构体（`Course`）**：存储课程编号、名称、学分、先修课列表、最早可排学期、实际排课学期，覆盖教学计划的核心属性。
- **辅助映射（`courseIndexMap`）**：通过`map<string, int>`建立“课程编号→数组索引”的映射，快速查询课程信息。
- **学期数据容器**：用`vector`存储各学期的已分配学分（`termCredits`）和课程列表（`termCourses`），便于动态维护排课状态。


#### 2. 关键逻辑设计
- **先修关系处理**：通过“最早可排学期”约束课程安排——每门课的最早学期 = 其所有先修课的最大学期 + 1（无先修课则为第1学期），若最早学期超过总学期数则直接判定排课无解。
- **排课策略实现**：
  - **学分均匀策略**：按“最早学期→先修课数量”排序课程，优先安排先修课多的课程，从最早学期开始尝试，尽量让各学期学分接近。
  - **课程集中策略**：按“最早学期→学分大小”排序课程，优先安排学分高的课程，从最早学期开始填充，尽量将课程集中在前期学期。


#### 3. 流程设计
```
输入课程数据 → 初始化（计算最早可排学期）→ 选择排课策略 → 执行排课逻辑 → 输出教学计划文件
```


### 三、功能模块说明
| 模块          | 功能描述                                                                 |
|---------------|--------------------------------------------------------------------------|
| 数据初始化模块 | 加载课程信息，计算每门课的“最早可排学期”，校验先修课合法性。             |
| 排课策略模块   | 实现“学分均匀”“课程集中”两种排课逻辑，自动匹配学期学分上限约束。         |
| 输出模块       | 将排课结果以“学期-课程-学分”的结构写入文件，便于查看与使用。             |

## 《教学计划编制程序》使用说明


### 一、环境准备
- 编译环境：支持C++11及以上标准的编译器（如GCC、VS2019+）。
- 运行方式：编译代码生成可执行文件后直接运行。


### 二、基础使用步骤
#### 1. 内置测试数据（默认）
程序`main`函数中已预置测试数据（6学期、12门课），直接运行即可体验：
- 编译代码：`g++ 教学计划.cpp -o 教学计划.exe`（Windows）或`g++ 教学计划.cpp -o 教学计划`（Linux）。
- 运行程序：执行可执行文件，选择排课策略（输入`1`或`2`）。
- 查看结果：程序自动生成`教学计划.txt`文件，包含各学期的课程、学分信息。


#### 2. 自定义课程数据（修改`main`函数）
若需替换为其他专业的课程信息，修改`main`函数中的以下部分：
```cpp
// 示例：修改总学期数、学分上限
totalTerms = 8;          // 总学期数
termCreditLimit = 12;    // 每学期学分上限

// 示例：修改课程列表（需保持数组长度一致）
vector<int> credits = {3,2,4,...};          // 课程学分列表
vector<string> courseIds = {"C01","C02",...}; // 课程编号列表
vector<string> courseNames = {"大学英语","Python基础",...}; // 课程名称列表
vector<vector<string>> preCourses = {{}, {"C01"},...}; // 先修课列表（无先修则为空数组）
```


### 三、排课策略选择
运行程序后，根据提示选择策略：
- 输入`1`：**学分均匀策略**——各学期学分尽量接近，适合希望学习负担平稳的需求。
- 输入`2`：**课程集中策略**——课程尽量集中在前期学期，适合后期有实习/科研安排的专业。


### 四、结果文件说明
`教学计划.txt`的内容格式示例：
```
===== 教学计划 =====
总学期数：6 | 每学期学分上限：10

第1学期（学分：5）：
  C1 - 程序设计基础（2学分）
  C9 - 高等数学（3学分）

第2学期（学分：6）：
  C2 - 高等数学（3学分）
  C4 - 汇编语言（3学分）
...
```
